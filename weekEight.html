JavaScript-Object Creation Functions, Inheritance, Properties, Methods, Insantiation

-Object literal-

   Creating an object in Javascript can be done as a 
literal. This is very similar to an associative array.
Just like with objects in other languages, we can use 
the "." operator to call member properties of the object.

EXAMPLE:
var student = 
{
	firstName: "Hayden", 
	lastName: "Carlson", 
	gender: "Male",
	year: "Senior",
	major: "Software Engineering"
};	

//Display object properties
    document.getElementById("lit").innerHTML = 
    "Name: " + student.firstName +
    " " + 
    student.lastName + 
    "Gender: " + student.gender +
    "Year: " + student.year  +
    "Major: " + student.major;

-Object using "new"-

	You can use the "new" keyword to create
the object. This lets you use the "." operator
to declare the member properties.

EXAMPLE: 
var studentOne = new Object();
studentOne.firstName = "Nathan";
studentOne.lastName = "Bassett";
studentOne.gender = "Male";
studentOne.year = "Sophomore";

//Display object properties
    document.getElementById("lit").innerHTML = 
    "Name: " + studentOne.firstName +
    " " + 
    studentOne.lastName + 
    "Gender: " + studentOne.gender +
    "Year: " + studentOne.year  +
    "Major: " + studentOne.major;

-Using a Constructor (ideal method)-

	Using a constructor is a more effective way to create
objects. This is similar to creating a class and then 
instansiating an object. You will be able to make multiple
objects this way.

EXAMPLE:
	function newStudent(fName, lName, gen, yr, mjr)
	{
		this.firstName = fName;
		this.lastName = lName;
		this.gender = gen;
		this.year = yr;
		this.major = mjr;
	}

	var studentTwo = new newStudent("Sean", "Carlson", "Male", "Sophomore", "Biology");
	var studentThree = new newStudent("Jacob", "Bassett", "Male", "Sophomore", "Web Design");

	//Display object properties studentTwo
    document.getElementById("constrOne").innerHTML = 
    "Name: "  + studentTwo.firstName +
    " " +
    studentTwo.lastName  + 
    "Gender: "  + studentTwo.gender  + 
    "Year: "  + studentTwo.year  + 
    "Major: "  + studentTwo.major;

    //Display object properties studentThree
    document.getElementById("constrTwo").innerHTML = 
    "Name: "  + studentThree.firstName + " " +
      studentThree.lastName  + " " +
    "Gender: "  + studentThree.gender  + " " +
    "Year: "  + studentThree.year  + " " +
    "Major: "  + studentThree.major  + " ";

-Inheritance-

	Objects inherit properties and methods from their
prototype (class definition). If you want to add a
property or a method to an object that was not defined
in the prototype, you would do the following:

studentThree.age = 26;

	This will add the "age" property to the object 
"studentThree" only. If you wish to add a new method 
to an object only, the following would suffice:

studentThree.fullName = function(){
	return this.firstName + " " + this.lastName;
};

	A way to add properties or method to a prototype,
the best way is to go into the prototype itselft and 
add the property or method. Another option would be
to use the prototype property. This will add
the property or method to the prototype. An example:

Property:
newStudent.prototype.age;

Method:
newStudent.prototype.fullName = funciton(){
	return this.firstName + " " + this.lastName;
};
