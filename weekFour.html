Variables

Declaring a variable is easy. Simply start
with 'var' then name the variable. Then assign the
variable a value such as '3'. Only when the
variable is assigned is it given a type.
Example:
var strVar = "This is a string variable.";
var numVar = 3;

Functions, Array's, Loops


An array is a collection of variables that
can be accessed. A function is a segment of
code that is set aside to perform a certain
task. A loop is a block of code that can be
executed multiple times.
Example:
function exampleFun(a){
var exampleArray = [1, 2, 3];
var answer = 0;
//loop
for (var i = 0; i < exampleArray.length; i++){
answer += exampleArray[i];
}

return answer + a;
}

document.getElementById("resultsTwo").innerHTML = exampleFun(4);
