Local Storage API, Storing and Receiving simple data

	You can store and receive data locally using the HTML 
local storage API. There are two ways to do this. The first
is with localStorage and the other is sessionStorage.
Local Storage
Session Storage 
	You can store local data as follows:
	   localStorage.setItem("firstname", "Hayden");

	And retreive it as follows:
	   document.getElementById("retreive").innerHTML = localStorage.getItem("firstname");

	You can also use the '.' operator:
	   localStorage.firstname = "Hayden";

	   and

	   document.getElementById("retreive").innerHTML = localStorage.firstname;
